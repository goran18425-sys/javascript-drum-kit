# JavaScript Drum Kit

A simple interactive drum kit built with JavaScript.

Press the buttons or keyboard keys to play different drum sounds.

## Features

- Interactive drum sounds
- Button and keyboard controls
- Simple and responsive interface

## Technologies

- HTML
- CSS
- JavaScript

## How to Use

1. Open the project in your browser
2. Click the drum buttons or press the corresponding keys
3. Play and experiment with the sounds
